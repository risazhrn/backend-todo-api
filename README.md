# 📝 Modern Todo List API

A modern, high-performance Todo List API built using Node.js, Fastify, TypeScript, and MongoDB.

## 🚀 Tech Stack
- **Runtime:** Node.js
- **Language:** TypeScript
- **Framework:** Fastify
- **Database:** MongoDB (via Mongoose)
- **Documentation:** Swagger / OpenAPI
